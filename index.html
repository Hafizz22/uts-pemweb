Belajar Pemrograman Web

Website ini dibuat untuk memenuhi Tugas UTS Pemrograman Web. Berisi pengenalan HTML, CSS, JavaScript, dan Tailwind; tata cara membuat website sederhana; serta kesan dan pesan belajar web.

1. Mengenal HTML, CSS, JavaScript & Tailwind

HTML

HTML itu bisa dianggap sebagai kerangka dasar dari sebuah website. Di dalamnya ada judul, paragraf, gambar, link, tabel, dan lain-lain. Tanpa HTML, halaman web tidak akan punya isi.

CSS

Kalau HTML sudah ada, tampilan biasanya masih polos. CSS lah yang bikin halaman jadi menarik. Misalnya mengubah warna, mengatur jarak, sampai memberi animasi.

JavaScript

JavaScript membuat website terasa hidup. Dengan JS, kita bisa bikin tombol yang berfungsi, menampilkan notifikasi, atau efek interaktif lain.

Tailwind CSS

Tailwind adalah framework CSS yang mempermudah styling. Kita cukup memanggil class yang sudah disediakan, tanpa harus menulis aturan CSS panjang-panjang.

2. Tata Cara Membuat Website Sederhana

Membuat website sederhana bisa dimulai hanya dengan satu file HTML. Berikut langkah-langkah yang biasanya dipakai mahasiswa pemula:

  1. Buat file baru bernama index.html
  2. Tulis struktur dasar HTML
  3. Tambahkan CSS atau Tailwind untuk desain
  4. Gunakan JavaScript jika ingin interaktif
  5. Buka file di browser untuk melihat hasilnya

Contoh kode sederhana:

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Website Pertama</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah website pertamaku.</p>
  </body>
</html>
